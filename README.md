# Etch-a-Sketch: <a href="isky-codes.github.io/etch-a-sketch/">Visit site here</a>

![Project Screenshot](https://github.com/Isky-Codes/Etch-a-Sketch/blob/main/Screenshot.png)

A repository to hold a simple Etch-A-Sketch app made from HTML, CSS, and Javascript. Users can input the length and height of the grid and hover over individual squares to highlight them in and create their own images like a real life Etch-A-Sketch.

# How It's Made:
Tech Used: HTML, CSS, Javascript
